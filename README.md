# moval

fast de duplication of different types of arrays


## Install

Install with npm:

```
$ npm install --save moval
```

## Usage

* require

```
const tools = require('moval');
```
* fast array de duplication

```
tools.Simplify(array)
```
* with a key value

```
tools.SimplifyByKey(array,'value')
```
